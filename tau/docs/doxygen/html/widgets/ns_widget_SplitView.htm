SplitView is a widget, which displays two children separated with a movable divider.
It allows to change the size ratio of its children and supports two orientations.

Options list

Option Input type Default value Description
data-divider-vertical boolean false specifies divider orientation
data-fixed boolean false if true, the divider can't be manipulated
data-ratio Array 0.5, 0.5]] width/height distribution between panes

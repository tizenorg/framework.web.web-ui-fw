Widget displaying loader popup on page changes.

Manual constructor

For manual creation of loader widget you can use constructor of widget:

var loader = ns.engine.instanceWidget(document.getElementById('ns-loader'), 'loader');

If jQuery library is loaded, its method can be used:

var loader = $('#ns-loader').loader();


$.mobile.loading( 'show', {
    text: 'foo',
    textVisible: true,
    theme: 'z',
    html: ""
});


$.mobile.loading('hide');

HTML Examples

Create simple loader pending from div using data-role:

<div data-role="loader"></div>

Options list

Option Input type Default value Description
data-html string '' custom html for the inner content of the loading messages
data-text string 'loading' the text to be displayed when the popup is shown
data-text-visible string false whether the text in the loading message is shown
data-theme string a the theme for the loading messages

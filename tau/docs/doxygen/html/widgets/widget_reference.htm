Widget Reference

The Tizen Web UI service provides rich Tizen widgets that are optimized for the Tizen Web browser. You can use the widgets for:

  • CSS animation
  • Rendering

The following table displays the widgets provided by the Tizen Web UI service.

Table: Tizen Widgets
Widget Description
Autodividers extension for ListView Widget The Autodividers widget extension automatically creates dividers for a ListView Widget.
Button Widget Button widget changes default browser buttons to special buttons with additional opportunities like icon, corners, shadow.
Checkboxradio widget Checkboxradio widget changes default browser checkboxes and radios to form more adapted to mobile environment(larger labels looks clickable)
Circularview widget
Collapsible widget
Collapsibleset widget
Controlgroup widget Controlgroup widget improves the styling of a group of buttons by grouping them to form a single block.
DateTimePicker widget
Dialog Widget
Fastscroll Widget
Field containers Fieldcontain widget improves the styling of labels and form elements on wider screens. It aligns the input and associated label side-by-side and breaks to stacked block-level elements below ~480px. Moreover, it adds a thin bottom border to act as a field separator.
Gallery widget
Listdivider widget Listdivider widget creates a list separator, which can be used for building grouping lists using {@link ns.widget.mobile.Listview Listview widget}.

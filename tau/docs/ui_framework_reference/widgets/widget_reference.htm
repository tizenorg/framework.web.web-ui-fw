Widget Reference

The Samsung Gear Web UI service provides rich Samsung Gear widgets that are optimized for the Samsung Gear Web application. You can use the widgets for:

  • CSS animation
  • Rendering

The following table displays the widgets provided by the Samsung Gear Web UI service.

Table: Samsung Gear Web UI Widgets
Widget Description
Button The button widget shows a control on the screen that can be used to generate an action event.
Checkboxradio The checkboxradio widget shows a list of options on the screen where one or more can be selected.
IndexScrollbar The indexScollbar widget shows an indexScrolblar having indices, usually for the list.
Toggle switch The toggle switch widget shows a 2-state switch on the screen.
Popup The popup widget displays a pop-up window.
Processing The progress widget shows that an operation is in progress.
Progress The progress bar widget shows a control that indicates the progress percentage of an on-going operation.
Slider The slider widget shows a control on the screen that you can use to change values by dragging a handle on a horizontal scale.

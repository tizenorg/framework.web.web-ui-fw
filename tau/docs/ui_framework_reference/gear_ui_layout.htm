Samsung Gear Web UI Framework Reference

Samsung Gear Web UI Framework is for the page navigation and the gear based appliaction development which is orgranized by themes.

Framework Services

Samsung Gear Web UI Framework consists of the following services

  • Page navigation

    To be able to navigate smoothly between pages for gear applications, there is a navigation javascript library available

  • Web Theme

    CSS themes and resources for Samsung Gear Web UI widgets (compatible with the widget source code) are available.

Limitations

The Samsung Gear Web UI framework has the following limitations:

  • The framework runs only on browsers supporting the HTML5/CSS standards. The draft version of the W3C specification is not fully supported.

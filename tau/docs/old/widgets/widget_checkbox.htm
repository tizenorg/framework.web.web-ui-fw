Checkboxradio

The checkbox widget shows a list of options on the screen where one or more can be selected.

To add a checkbox widget to the application, use the following code:

<input type="checkbox" name="mycheck" id="check-test"  />
<label for="check-test">Checkbox</label>

To add a radio widget to the application, use the following code:

<input type="radio" name="radioset" id="radio-1" />
<label for="radio-1" class="ui-popup-radio-label">Radio</label>

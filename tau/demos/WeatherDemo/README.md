========================================
Weather demo prototype for multi-devices
========================================

This demo is a prototype that shows how a single html is compatible with a varierty of devices and can be optimized for any screen resolution.

It supports the following targets below.

	- W device
	- Mobile ( landscape, portrait )
	- Tab ( landscape, portrait )

This demo has the following files below.

	- index.html
	- stylew.css : style sheet for index.html
	- swipe.js : support the swipe gesture for W device
